# Repo Status

`ASAD` is currently the backend prototype repo.

It is no longer the primary source of truth for shared product planning.

Use `blank-canvas` as the shared repo for:

- frontend work
- contracts
- coordination docs
- handoff instructions

Use `ASAD` for:

- backend experiments
- renderer prototyping
- prompt and pipeline prototyping

If a backend piece becomes stable enough to keep, migrate it into the shared product repo rather than duplicating long-term planning across both repositories.
